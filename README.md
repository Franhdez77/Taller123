ATM WITHDRAWAL SIMULATOR

This is a simple Python program that simulates a basic ATM Withdrawal system. The program starts with an initial account balance and allows the user to withdraw money while checking for invalid or insufficient amounts.

1- The program display a welcome message and shows the account balance. 
2- The program allows the user to input the amount. 
3- The program shows error mesagges in case that the amount to withdraw are 0, negative number or bigger than initial value
4- If the amount is in the correct range, the program display a succesfully message, and this shows the new value in the account
